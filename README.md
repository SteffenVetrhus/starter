# Template App

A minimal React Router v7 template with TailwindCSS.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit `http://localhost:5173`

## Build

```bash
npm run build
```

## Production

```bash
npm run start
```

## Docker

```bash
docker build -t template-app .
docker run -p 3000:3000 template-app
```

## Stack

- React Router v7
- React 19
- TypeScript
- TailwindCSS v3
- Vite

Start building!
